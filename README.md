# 介绍
  金猫wms 是基于vue2 + iview2 + axios + webpack2 为主要技术，开发的一个后台管理系统。</br>
  
## 安装
  可以有多种方法进行安装。
### 使用命令行
  github</br>
* Fork 当前库
* 复制一下命令到您的目录</br>
```
https://github.com/hanjiangxueying/vue2-iview2-admin.git
```
## 浏览器支持
* IE 9+
* Firefox（最新）
* Chrome（最新）
* Safari（最新）
## 运行项目
  进入vue2-iview2-admin 运行一下命令安装依赖包。
```
npm install
```
  安装完成之后，开始运行项目。
```
npm run dev 
```
## 编译项目
  运行一下命令：
```
npm run build
```
